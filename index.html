🔮 El Método Mágico

5 pasos para convertir cualquier decimal periódico mixto a fracción

1

🔍 Identifica las partes

Un decimal periódico mixto tiene una parte no-periódica y una que se repite.

Ejemplo: 2.345
Parte entera: 2
No periódica (n=1 cifra): 3
Período (p=2 cifras): 45
2

🔑 El truco del denominador

El denominador se construye con 9s y 0s, ¡sin hacer ninguna multiplicación!

👉 Escribe un 9 por cada cifra del período
👉 Luego un 0 por cada cifra no periódica

Ejemplo: 2.345
Período = 2 cifras → 99
No periódica = 1 cifra → 0
Denominador = 990 = 990 ✅
3

🔢 Calcula el numerador

El numerador es la diferencia entre dos números enteros (sin la coma decimal):

Numerador = (todo el número sin coma) − (parte entera + no periódica)

Ejemplo: 2.345…
Todo sin coma → 2345
Entera + no periódica → 23
Numerador = 2345 − 23 = 2322
4

🧩 Arma la fracción

Numerador = 2322
Denominador = 990 = 990

Fracción sin simplificar: 2322 / 990
5

✂️ Simplifica con el MCD

MCD(2322, 990) = 18
2322 ÷ 18 = 129  |  990 ÷ 18 = 55
Resultado: 129 / 55 ✅
⚡ REGLA MÁGICA ⚡
📝 Numerador = todo sin coma − (entera + no periódica)
🔑 Denominador = un 9 por cada cifra del período + un 0 por cada cifra no periódica
